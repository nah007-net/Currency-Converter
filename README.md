# Currency-Converter
A hands-on project developed for the Object-Oriented Programming course at UNINTER.


(●'◡'●) About

A virtual piggy bank simulator where the user can add and remove coins from different countries, list the stored currencies, and calculate the total converted to Brazilian Reais (BRL).


(^///^) Class Structure
Moeda (abstract)


├── Real
├── Dolar
├── Euro
└── Iene

Cofrinho       → manages the currency collection (ArrayList<Moeda>)

Principal      → entry point, interactive terminal menu


(/≧▽≦)/ Features

Add currencies to the piggy bank (BRL, USD, EUR or JPY)

Remove currencies from the piggy bank

List all stored currencies

Calculate the total value converted to Brazilian Reais

 
Exchange Rates

CurrencyRate (in BRL)Real (BRL)1.00Dollar (USD)5.40Euro (EUR)6.20Yen (JPY)0.035

Rates are hardcoded for academic purposes.

\^o^/ Tech Stack

Java (Eclipse IDE)

Paradigm: Object-Oriented Programming

Concepts applied: inheritance, abstraction, polymorphism, encapsulation
